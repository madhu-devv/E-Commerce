# Ecommerce Website

An e-commerce web application built using React. The project fetches product data from the DummyJSON API and includes shopping cart and search functionality.

## Features

- Product listing using DummyJSON API
- Shopping cart functionality
- Product search functionality
- Responsive UI

## Tech Stack

- React
- JavaScript
- CSS
- DummyJSON API

## Future Improvements

- Add user authentication
- Store cart data using LocalStorage
- Add product filtering and sorting
- Improve mobile responsiveness
